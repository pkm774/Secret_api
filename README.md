# Secret API

A **simple** web application developed with the use of **FrontEnd**, **BackEnd** & **REST APIs**.

## Technologies used

  * HTML, **EJS (Embedded Java Script), JQuery**
  * CSS
  * Javascript, **Express.js**
  * **Nodejs, npm**
  * **REST APIs**

## Features

The application uses **GET**, **POST**, **PUT**. **PATCH** & **DELETE** HTTP Methods to fetch, update and delete secrets and score using id fetched from [secrets-api-appbrewery](https://secrets-api.appbrewery.com).
